# Secure Media Encryption System (AES-GCM)

This project is an experimental implementation of end-to-end encrypted media handling using AES-256-GCM.

## Features
- AES-256-GCM authenticated encryption
- Image and video file encryption/decryption
- Secure nonce and tag handling
- Certificate and TLS experiments
- Early QR-based onboarding concept

## Tech Stack
- Python
- PyCryptodome
- OpenSSL
- Flask (experiments)

## Disclaimer
This project is for academic and learning purposes.
Sensitive files (keys, certificates, media) are excluded from the repository.
